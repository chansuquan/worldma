# Worldmap-Git-User
A very simple boilerplate to get started to development.

### Features
- React
- Express
- React-Router
- ES6,ES7 Support
- CSS-Loaders
- Webpack
- Babel
- React-Simple-Maps
#### Installation
```
$ git clone https://github.com/chansuquan/worldmap-git-users.git <YOUR_PROJECT_NAME>
$ cd <YOUR_PROJECT_NAME>
$ npm install
```
#### Development
Run the development server:
```
$ npm run dev
```
#### Production
Build and run the production server:
```
$ npm run build
$ npm start
```
